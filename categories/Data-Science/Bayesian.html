Framework

The Bayesian framework is based on conditional probabilities.

P(A∣X)=P(X∣A)×P(A)P(X)P(A|X) = \frac{P(X|A)\times P(A)}{P(X)}P(A∣X)=P(X)P(X∣A)×P(A)​

Posterior probabilities lead us to update our theory.

The wider the curve, the greater the uncertainty.

Often, we plot the prior probability vs the posterior probability. The x value is the initial probability, and the y value is our updated probability.

Probability Distributions

Discrete

For some discrete random variable ZZZ, there is a pprobaility mass function P(Z=k)P(Z=k)P(Z=k) which maps kkk to a probability. k∈Nk \in \Nk∈N because it is discrete.

Discrete variables follow the Poisson distribution.

P(Z=k)=λke−λk!,    k=0,1,2,…P(Z = k) =\frac{ \lambda^k e^{-\lambda} }{k!}, \; \; k=0,1,2, \dotsP(Z=k)=k!λke−λ​,k=0,1,2,…

Poisson

Denoted Z∼Poi(λ)Z \sim \text{Poi}(\lambda)Z∼Poi(λ)

Properties:

  • λ\lambdaλ is the intensity of the distribution. Patterns of lambda include:
    • Increasing means bigger values will give bigger probabilities
      • Decreasing means smaller values will give bigger probabilities
  • The expected value is equal to the parameter.

E[  Z  ∣  λ  ]=λE\large[ \;Z\; | \; \lambda \;\large] = \lambdaE[Z∣λ]=λ

Continuous

Follows a probability density function.

fZ(z∣λ)=λe−λz,    z≥0f_Z(z | \lambda) = \lambda e^{-\lambda z }, \;\; z\ge 0fZ​(z∣λ)=λe−λz,z≥0

A common case is the exponential distribution

Z∼Exp(λ)Z \sim \text{Exp}(\lambda)Z∼Exp(λ)

Properties

We have the expected value identity

E[  Z  ∣  λ  ]=1λE[\; Z \;|\; \lambda \;] = \frac{1}{\lambda}E[Z∣λ]=λ1​

What does this mean in English? For some random variable given some coefficient, the expected value of that random variable is equal to the inverse of the coefficient.

Solving for Lambda

Switchpoints

Consider graphing rain in Seattle. Some spans of days are dry, and then all the sudden, it spikes. One might model λ\lambdaλ with a piecewise function based on a switchpoint.

λ={λ1if t<τλ2if t≥τ\lambda = \begin{cases} \lambda_1 & \text{if } t \lt \tau \cr \lambda_2 & \text{if } t \ge \tau \end{cases}λ={λ1​λ2​​if t<τif t≥τ​

Here, τ\tauτ is the switchpoint. We can assign a uniform density function to it, that is, it could be on any one of the 70 days.

Lambda's Continuity

Lambda can be any positive number. Therefore, we need to model it with a PDF.

The exponential density function takes a λ\lambdaλ of it's own. Thus, we need to sub in another variable for that because we're trying to solve for λ\lambdaλ

λ1∼Exp(α)λ2∼Exp(α)\lambda_{1} \sim \text{Exp}(\alpha) \\ \lambda_{2} \sim \text{Exp}(\alpha)λ1​∼Exp(α)λ2​∼Exp(α)

Hyperparameters

Here, α\alphaα is the hyperparameter, a parameter defining another variable. Generally, these have little effect on the outcome as a whole.

Good practice is to set α\alphaα to the inverse of the data's mean.
